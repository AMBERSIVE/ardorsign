# Ardorsign

## Install

```bash
npm install ardorsign -g
```

## Useage

This is a command line tool to sign transactions via cli.
To sign a transaction execute the following command:

```bash
ardorsign --transaction='ENTER TRANSACTION BYTES' --secret='SECRET'
```
